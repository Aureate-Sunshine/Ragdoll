## Extension Settings

插件设置项:

- `ragdoll-yapi.yapi.ldapLogin`: 启用 LDAP 账号登录
- `ragdoll-yapi.yapi.email`: 登录用户名
- `ragdoll-yapi.yapi.password`: 登录密码
- `ragdoll-yapi.yapi.url`: Yapi 项目地址
- `ragdoll-yapi.yapi.pid`: Yapi 项目 ID
